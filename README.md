# Node Restful API

An API built using NodeJS, Express and MongoDB

# Process
- Initialise Node app
```
npm init
```
- Install Express
```
npm install express --save
```
- Create index.js at application route
```
touch index.js
```
- Add GitHub origin
```
git remote add origin https://github.com/geejay81/node-restful-api.git
git push -u origin master
```
